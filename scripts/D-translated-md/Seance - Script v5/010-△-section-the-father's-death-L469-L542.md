# △ 章節：父親的死亡

\+ 地點

**（靈界）** 在玩家的左側有一座祭壇。中央擺放著林山德的靈堂遺像（描繪著哀傷、孤寂的表情）。除了兩側的花束外，祭壇上堆滿了銅錢和金銀首飾。玩家正前方是一個棺材，棺蓋未完全蓋上，留有一條狹窄的縫隙。

\+ 旁白

**（幽靈合唱團）** 打開封印 / 解開他們的嘴巴

\+ 說明  
_牆上可見三張照片，每張都蓋有能量封印。當玩家注視照片時，照片開始動起來並說話，我們聽到其中一個孩子的聲音，說著正面且善良的話語。玩家使用相機拍攝並收集後，封印被解除，孩子們的話語變得尖酸刻薄…  
_

\+ 旁白

**（文宗）** 多麼盡責的父親！也是個好丈夫，直到妻子去世……他只愛錢。他只想要無盡的財富……

\+ 旁白

**（春花）** 多麼堅強的男人！多麼有原則！……他不讓我結婚。當我違抗他時，他當然會來，毀掉婚禮……

\+ 旁白

**（慧）** 多麼冷靜。多麼沉著……他的脾氣很兇。他毫不留情！……

\+ 說明  
_收集完三段對話後，棺材突然震動。_ _牆上的黑影消失，燈光變暗。數束聚光燈聚焦在棺材上。  
_

\+ 旁白

**（幽靈合唱團）** 死了 / 沒死

\+ 互動  
玩家打開棺材蓋。（互動選項：1. 玩家站起來，走向棺材，推開棺蓋。2. 玩家保持坐姿，使用相機強行打開。3. 玩家不動，棺蓋在設定時間後自動滑開。）△ 棺材打開後，林山德躺在裡面，雙眼閉著，手死死抓著帳本。

\+ 旁白

**（幽靈合唱團）** 沒死 / 有呼吸

\+ 互動  
玩家與父親互動。（互動選項：1. 玩家站起來，伸手觸摸他的胸口。2. 玩家保持坐姿，使用相機；林山德的眼睛突然睜開。3. 玩家不動，林山德的眼睛在設定時間後自動睜開。）

\+ 聲音捕捉（動作）

**（父親）** 林山德的眼睛突然睜開。他的目光迷離，伸手摸向胸前的墜飾——卻發現不見了。突然，林山德的瞳孔轉動，終於聚焦在玩家的臉上。

\+ 聲音捕捉（對話）

**（父親）** 你是誰？你在這裡做什麼，間諜？小偷！

\+ 聲音捕捉（動作）

**（父親）** 林山德怒吼著，猛然坐起，想要趕走玩家。

\+ 說明  
_棺材蓋重重地砰然關上。  
_

\+ 旁白

**（幽靈合唱團）** 可惜 / 秘密 / 在哪裡

\+ 互動  
祭壇上的靈堂遺像開始晃動。遺像上出現了能量封印。玩家與林山德的靈堂遺像互動，使用相機捕捉能量封印。

\+ 影片

**（父親）** 互動後，林山德遺像的表情從哀傷孤寂轉為憤怒，目光怒視房間的某個角落／牆壁。

\+ 互動  
玩家舉起相機，對準林山德注視的方向按下快門。一道通往下一個靈界的隱藏入口顯現。
