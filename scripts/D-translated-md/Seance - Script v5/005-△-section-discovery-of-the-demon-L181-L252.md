# △ 章節：惡魔的發現

\+ 說明  
_在結界建立後，一股詭異的黑霧開始纏繞在林淑美身上。_  

\+ 旁白

**(幽靈合唱團)** 攝影機 / 看見 / 收集！  

\+ 互動  
玩家被引導去拿起攝影機。  

\+ 說明

**(委託人)** _林淑美好奇又興奮地盯著攝影機。_  

\+ 角色語音（對話）

**(委託人)** 這就是傳說中的靈魂相機嗎？我……能看到鬼魂嗎？我好想再見到我的家人……  

\+ 說明

**(委託人)** _林淑美再次露出擔憂的神情。_  

\+ 互動  
玩家對林淑美拍照。  

\+ 角色語音（對話）

\+ 情緒：（困惑）

**(委託人)** 你為什麼要拍我？  

\+ 角色語音（對話）

\+ 情緒：（驚恐）

**(委託人)** 惡靈！果然是真的！我被詛咒了！  

\+ 互動  
玩家使用攝影機對準林淑美，試圖捕捉能量（黑霧）。拍攝後，玩家獲得林淑美的照片，但她身上的黑霧並未減少，儘管水晶已充滿黑色能量。  

\+ 旁白

**(幽靈合唱團)** 更多！/ 有強烈的東西！/ 裡面……  

\+ 角色語音（對話）

**(委託人)** 你拍到我好看的角度了嗎？  

\+ 角色動作

**(委託人)** _林淑美放鬆下來。除了被拍照外，什麼事都沒發生。她擺出姿勢，用雙手框住臉，斜靠在椅子上。沒有比「V」的手勢！_  

\+ 互動  
玩家拍攝足夠多的林淑美照片，直到水晶完全充滿。水晶充滿後，攝影機螢幕變成紅色。透過鏡頭，在林淑美身後出現一個漆黑的人影。那影子的雙手放在她肩膀上，就像操縱木偶的傀儡師。臉部被陰影遮蔽。  

\+ 角色動作

林淑美被操縱成木偶的畫面在攝影機螢幕上清晰可見。  

\+ 旁白

**(幽靈合唱團)** 靈體附身！/ 被附身！/ 被控制！  

\+ 角色動作

**(委託人)** 林淑美同時發出愉悅的笑聲。
