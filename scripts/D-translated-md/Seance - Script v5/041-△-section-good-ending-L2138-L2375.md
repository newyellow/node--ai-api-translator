# △ 章節：美好結局

\+ 說明  
_玩家握著林淑美的手。_

\+ 動作捕捉 (Action)

**(客戶)** 林淑美抽搐扭動，玩家無法鬆手。最後，她飛回椅子上。  

\+ 旁白

**(惡魔)** 不！我不會成為你的傀儡！不會再這樣了！不！  

\+ 說明  
_光芒逐漸消退，惡魔的身影被吸入玩家自己的手中。_  

\+ 動作捕捉 (Action)

**(客戶)** 房間恢復寂靜。林淑美的身體癱倒在椅子上，喘息著。  

\+ 旁白

**(幽靈合唱團)** 封印 / 墓葬 / 幸運…  

\+ 動作捕捉 (Action)

**(客戶)** 過了一會兒，林淑美睜開眼睛。  

\+ 對話捕捉 (Dialog)

**(客戶)** 哎呀…我…我在哪裡…？  

\+ 動作捕捉 (Action)

**(客戶)** 她緊張地四處張望。  

\+ 對話捕捉 (Dialog)

**(客戶)** 你是誰？我在哪裡？它讓我…但…它現在不見了？它陪伴我這麼久——現在卻消失了。  

\+ 說明

房間邊緣的蠟燭開始閃爍。  

\+ 對話捕捉 (Dialog)

**(客戶)** 還有別人在這裡…我能感覺到。  

\+ 旁白

**(幽靈合唱團)** 蠟燭 / 光 / 聯繫  

\+ 互動  
玩家點燃一根蠟燭。火焰最初微弱且閃爍不定。玩家必須用雙手捧住火焰以穩定它。  

\+ 說明

_火焰穩定後，一張寫有「囍」字的紙片從天花板飄落，落在玩家面前的桌上。紙片一面燒焦。_  

\+ 對話捕捉 (Dialog)

**(客戶)** 這是一份供品。獻給火焰。  

_\+_ 互動

玩家將紙片放入蠟燭中燃燒，火焰變得明亮，逐漸顯現出林春華的身影。  

\+ 對話捕捉 (Dialog)

**(姊姊)** 妹妹。  

\+ 動作捕捉 (Action)

**(客戶)** 林淑美凝視著微笑的林春華。  

\+ 對話捕捉 (Dialog)

**(姊姊)** 妹妹，你做得很好。  

\+ 對話捕捉 (Dialog)

情緒：（不理解）

**(客戶)** 我做了什麼？  

\+ 對話捕捉 (Dialog)

**(姊姊)** 你抵抗那個…東西…這麼久。它輕易奪走了我們其他人，但你堅持了這麼久。靈力在你身上很強，淑美…  

\+ 動作捕捉 (Action)

**(客戶)** 林淑美開始顫抖。  

\+ 對話捕捉 (Dialog)

**(客戶)** 你穿著婚紗真美，姊姊。  

\+ 說明：

如果玩家選擇使用阿輝的幽靈：

\+ 對話捕捉 (Dialog)

**(客戶)** 但阿輝…他走了。惡魔帶走了他。  

\+ 說明：

如果玩家選擇使用林文宗的幽靈：

\+ 對話捕捉 (Dialog)

**(客戶)** 你可以回去找阿輝。他現在準備好了。  

\+ 說明：

無論哪種情況，我們在此處會合。  

\+ 對話捕捉 (Dialog)

**(姊姊)** 我知道。事情就該如此，妹妹。別難過。惡魔已離開我們家族。這個靈媒將承擔這份重擔。  

\+ 動作捕捉 (Action)

**(客戶)** 林淑美深吸一口氣，努力消化巨大的悲傷。過了一會兒，她擦去眼淚。  

\+ 對話捕捉 (Dialog)

**(客戶)** 現在我該怎麼辦？  

\+ 對話捕捉 (Dialog)

**(姊姊)** 隨你選擇！  

\+ 動作捕捉 (Action)

**(姊姊)** 燃燒的蠟燭發出耀眼光芒。林春華的身影開始變得透明，周圍漂浮著光點。  

\+ 對話捕捉 (Dialog)

**(姊姊)** 我必須走了。  

\+ 對話捕捉 (Dialog)

**(客戶)** 春華…  

\+ 動作捕捉 (Action)

**(姊姊, 客戶)** 他們短暫擁抱，但林春華先鬆開。  

\+ 說明

_如果玩家犧牲了林文宗的幽靈：_

\+ 對話捕捉 (Dialog)

**(姊姊)** 再見了，小兔子。我即將結婚！  

\+ 動作捕捉 (Action)

**(養子)** 阿輝出現，攬著林春華的肩膀。  

\+ 對話捕捉 (Dialog)

**(養子)** 謝謝你，妹妹，祝你好運。你給了我們超乎想像的東西。  

\+ 說明

_如果玩家犧牲了阿輝的幽靈：_

\+ 對話捕捉 (Dialog)

**(姊姊)** 再見了，小兔子。我將去找文宗。  

\+ 動作捕捉 (Action)

**(兄弟)** 林文宗出現在林春華身旁。  

\+ 對話捕捉 (Dialog)

**(兄弟)** 我們會守護你，親愛的姊姊。你有我們的祝福。  

\+ 說明

_分支在此合流。_

\+ 動作捕捉 (Action)

**(客戶)** 林淑美點頭，悲傷明顯。  

\+ 對話捕捉 (Dialog)

**(客戶)** 別走…別離開我…  

\+ 說明  
_林春華的身影漂浮著，半透明消散中_  

\+ 旁白

**(幽靈合唱團)** 送她回家！/ 鈴鐺！  

\+ 互動：

玩家搖響銀鈴。  

\+ 說明

_林春華化為無數金色光點，像螢火蟲般飄向空中，隨後消失。_  

\+ 動作捕捉 (Action)

**(客戶)** 目送姊姊離去，帶著悲傷與驚嘆。這一幕很美。  

\+ 對話捕捉 (Dialog)

**(客戶)** 我不再害怕了，靈媒。我感覺靈魂就在身邊…你會…你會教我嗎？  

\+ 動作捕捉 (Action)

**(客戶)** 她伸出手，縮回，再次伸出。  

\+ 互動

玩家觸碰她的手。  

\+ 旁白

**(幽靈合唱團)** 安全 / 好女孩 / 自由  

\+ 互動：

玩家伸出手——封印仍刻在上面，散發著強烈的內光…
