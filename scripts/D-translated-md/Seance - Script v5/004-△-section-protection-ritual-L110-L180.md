# △ 區段：保護儀式

\+ 說明  
_林淑美環顧四周，現在緊張感稍減。_

\+ 角色語音 (對話)

**（委託人）** 這間房間就是你工作的地方嗎？看起來並不那麼可怕。我…我可以坐下嗎？謝謝。  

\+ 說明  
_林淑美坐在桌子對面。_  

\+ 角色語音 (對話)

**（委託人）** 我們要開始了嗎？我帶了東西，像你說的那樣。每個人的紀念物。對吧？  

\+ 說明  
_玩家點頭肯定。林淑美點頭，神情中帶著奇異的興奮與期待。_  

\+ 角色語音 (對話)

**（委託人）** 那我準備好了。我很害怕…但我相信你。你會保護我，對吧？  

\+ 旁白

**（幽靈合唱）** 鹽 / 保護  

\+ 互動  
玩家將鹽撒在桌子上。  

\+ 說明

**（委託人）** _當鹽被撒出時，林淑美帶著極大的興趣和好奇心注視著這個過程。_  

\+ 角色語音 (對話)

**（委託人）** 我家以前是做鹽生意的，但那並沒有救我們…  

\+ 旁白

**（幽靈合唱）** 蠟燭 / 點亮！/ 驅散黑暗！  

\+ 互動  
玩家點燃桌上的所有蠟燭。  

\+ 旁白

**（幽靈合唱）** 血 / 針！  

\+ 互動  
玩家刺破手指，滴血在鹽上。  

\+ 角色語音 (對話)

**（委託人）** 哎呀！  

\+ 說明  
_當血滴落在鹽上時，出現一道保護屏障，包圍著玩家、桌子和委託人。_  

\+ 說明

**（委託人）** _委託人帶著興趣看著包圍他們的屏障。這明顯不夠堅固，看起來脆弱、易碎。她卻顯得異常自信；既不驚訝也不感到佩服。_  

\+ 旁白

**（幽靈合唱）** 脆弱 / 不夠 / 危險！
